Project Overview

NexaSec is an advanced cybersecurity solution designed for threat detection, vulnerability assessment, and network security monitoring. It features a FastAPI backend and a Next.js frontend, providing businesses with real-time security insights and protection.

Key Features

Real-time Threat Detection

Vulnerability Assessment

Network Security

AI-powered Intelligence

Architecture

Backend

  • FastAPI-based RESTful API
  • MongoDB with Motor
  • WebSocket Integration
  • Microservices Architecture

Frontend

  • Next.js 13+ with App Router
  • React with TypeScript
  • Real-time Visualization
  • Tailwind CSS

Technology Stack

Backend Technologies

  • Python 3.8+

Installation Guide

Backend Setup

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

Environment Configuration

MONGODB_URL=your_mongodb_url
JWT_SECRET=your_jwt_secret
JWT_ALGORITHM=HS256

Frontend Setup

npm install

Environment Configuration

NEXT_PUBLIC_API_URL=http://localhost:8000

Documentation

API Endpoints

Authentication

  • POST /auth/login
  • POST /auth/signup
  • POST /auth/refresh
  • POST /auth/logout

Vulnerability Management

  • GET /api/vulnerabilities
  • POST /api/vulnerabilities/scan
  • PATCH /api/vulnerabilities/{id}/status

Network Security

  • GET /api/network/devices
  • POST /api/network/scan
  • GET /api/network/threats
